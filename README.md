# Ynov Swift

Projet Swift pour exercices et TP Ynov.

## Prerequis

- Swift 5.9+
- Windows 10/11, macOS ou Linux

### Installation Swift

**Windows :**
```bash
winget install Swift.Toolchain
```

**macOS :**
```bash
xcode-select --install
```

**Linux :**
```bash
sudo apt install swift
```

## Lancer le projet

```bash
cd MyExecutable
swift run
```
